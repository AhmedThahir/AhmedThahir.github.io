# Welcome!

This is a PWA using Mkdocs material theme

## Performance Optimizations

- Network-first PWA (offline-use possible)
- Minifying & combining CSS/JS
- Lazy loading images
- Prefetching links
- Using katex instead of mathjax
- `content-visibility: auto`
