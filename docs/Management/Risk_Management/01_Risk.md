# Axes

- Awareness Type
- Scope Type

## Awareness Type

| Type            | Fancy Name                                      | Cause                                                                                                  | Modellable                           | Compensation expected       |
| --------------- | ----------------------------------------------- | ------------------------------------------------------------------------------------------------------ | ------------------------------------ | --------------------------- |
| Known known     | Probabilistic risk/<br>Aleatoric Risk           | - Randomness in process<br>- Measurement errors                                                        | ❌                                    | ✅                           |
| Unknown known   | Hidden knowledge/<br>Latent Risk                | - Model misspecification<br>- Did not incorporate into model<br>- Maybe others know but you don’t know | ✅<br>Validation<br>Scenario planning | ✅<br>Comparing to benchmark |
| Known unknown   | Epistemic Risk                                  | Insufficient data                                                                                      | ✅<br>Further research                | ❌                           |
| Unknown unknown | Uncertainty/<br>Ontological risk/<br>Black swan | Completely unexpected/unforeseeable events                                                             | ✅<br>Further research                | ❌                           |

## Scope Type

| Type         |                                                                 | Minimizable<br>for Action if not modelled |

| Systematic   | - External/Macro<br>- Sensitivity to global/market fluctuations | ❌                                         |
| Unsystematic | - Internal/Micro<br>- Individual factors                        | ✅<br>through diversification (portfolio)  |
